# ⚡ PowerManager

> macOS menu bar app for battery optimization — Intel & Apple Silicon (M1/M2/M3)

## Features

- 🔋 **3 Power Profiles** — Power Save, Balanced, Performance
- 🌡️ **Hardware Monitoring** — CPU temperature, frequency, load, RAM usage, fan speed, watts
- 🏥 **Battery Health** — cycle count, condition, health percentage
- 🌍 **12 Languages** — English, Српски, Hrvatski, Bosanski, Русский, Українська, Deutsch, Français, Español, Italiano, Polski, Türkçe
- ⏱️ **15-day Free Trial** — no credit card required
- 🔒 **License System** — machine-locked serial numbers

## Requirements

- macOS 12 (Monterey) or later
- Intel or Apple Silicon Mac (Universal Binary)

## Installation

1. Download `PowerManager.dmg` from [Releases](../../releases)
2. Open the DMG and drag PowerManager to Applications
3. Launch the app — click Allow when asked for hardware monitoring setup (one-time)
4. Select your preferred language on first launch
